    Welcome to my simple static website, hosted on AWS. As an IT professional—and especially as an aspiring cybersecurity specialist—I believe it's important to get hands-on experience with a variety of technologies and platforms. So I built this site to dive into HTML, CSS, JavaScript, and AWS services.



    Through this project, I’ve gained practical experience with AWS CloudFront and S3, ensuring that static site content is served securely over HTTPS. I’ve also configured a custom domain using Route 53 to make the site more accessible and professional. Since it’s a static site, I manage updates using Git and track all changes in a GitHub repository. This project helped me get comfortable with version control and the AWS CLI on my local machine.
